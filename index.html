About Me

Hi there! I'm Connie. I'm a full-stack software engineer based in NYC.

My first experience with code was building website templates with simple HTML and CSS when I was 12 to keep myself engaged during the summer. At the time, I didn't know what programming was.

I found excitement in Math so naturally it became what I chose to study in college, with the intention of pursuing a second bachelor's degree in Nursing.

During my experience in the healthcare industry as a nursing assistant, while exploring the features of various medical software applications I was expected to become acquainted with, my curiosity in tech resurfaced. From there, I made the decision to dive into the world of programming and was drawn towards my passion for problem solving. Recently, I've completed an immersive software engineering bootcamp at The Grace Hopper Program at Fullstack Academy.

When I'm not coding, I'm either baking, trying new food or visiting aquariums!

Technical Projects

Codescape

An educational platform designed with Escape the Room game logic for practicing fundamental JavaScript concepts deployed on Heroku. Players must solve challenges (levels ranging from easy, medium to hard) to unlock items to escape the room.

Technologies - JavaScript, React, Redux, Node.js, Express, Sequelize, PostgreSQL, Docker, Mocha, Chai.

Let It Brie

An e-commerce website that sells charcuterie boards, meats, and cheeses deployed on Heroku. Implemented cart functionality which persists over sessions using local storage and developed backend to post charges to Stripe.

Technologies - JavaScript, React, Redux, Node.js, Express, Sequelize, PostgreSQL, Google OAuth, Stripe API.
